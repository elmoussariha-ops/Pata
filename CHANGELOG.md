# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project follows [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-04-13

### Added
- Rust workspace with modular crates for traits, memory, reasoning, orchestration, personas, CLI, and HTTP server.
- Multi-persona support (`developer`, `teacher`, `personal`, `smb`) through a registry-based architecture.
- Deterministic reasoning and verification pipeline with structured traces and confidence outputs.
- Runnable quickstart demo and public benchmark scripts for reproducible evaluation and sharing.
- Public launch and product documentation (overview, roadmap, launch pack, use-case proofs).

### Quality
- Main-branch CI with formatting, strict clippy, and workspace test checks.
